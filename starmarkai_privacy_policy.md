# Privacy Policy for StarMarkAI Chat Bookmarks

**Effective Date:** May 9, 2026

StarMarkAI Chat Bookmarks helps users bookmark, organize, search, and return to specific sections of supported AI chat pages such as ChatGPT and Claude.

## Data We Handle

StarMarkAI may store the following data when you use the extension:

- User-created chat bookmarks
- Short chat snippets, capped to 150 characters
- Chat URLs for saved bookmarks
- Bookmark timestamps
- Project and chat display names
- User-edited bookmark, chat, or project labels
- Theme, sorting, search, and sync preferences
- Optional JSON backup data when you choose to use JSON backup
- Optional Chrome Sync data when you enable Chrome Sync

## How Data Is Used

This data is used only to provide the extension's core features:

- Saving bookmarked chat sections
- Showing bookmarks in the side panel
- Searching and sorting bookmarks
- Jumping back to saved chat sections
- Syncing bookmarks across signed-in Chrome browsers when Chrome Sync is enabled
- Creating or restoring JSON backups when you choose to use JSON backup

## Data Storage

By default, StarMarkAI stores data locally in Chrome extension storage on your device.

If you enable Chrome Sync, bookmark data may sync through your signed-in Chrome account using Chrome's built-in sync features.

If you use JSON backup, the backup file is created, selected, and controlled by you.

## Data Sharing

StarMarkAI does not sell, rent, share, or transfer your data to third parties.

StarMarkAI does not send your bookmark data to our own servers.

## Remote Code

StarMarkAI does not load remote JavaScript or WebAssembly. All code is included in the extension package.

## Supported Websites

StarMarkAI runs only on supported AI chat websites needed for its functionality, such as ChatGPT and Claude. It reads page content only to let you bookmark selected chat sections and return to them later.

## Data Retention and Deletion

You can delete bookmarks from the extension at any time.

You can also clear the extension's stored data from Chrome extension settings or by uninstalling the extension. If Chrome Sync is enabled, synced data may remain associated with your Chrome account until cleared through Chrome's sync mechanisms.

## Changes to This Policy

This privacy policy may be updated if StarMarkAI's functionality changes. Updates will be reflected by changing the effective date above.
